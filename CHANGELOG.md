# Changelog

## v0.1.0 - 2026-05-01

- Initial public release of the WPV1 phylogeographic analysis template.
- Added a synthetic example dataset for reproducible trial runs.
- Added distance-tree, FastTree, and IQ-TREE execution modes.
- Added transition summaries, lineage summaries, root-to-tip diagnostics, and optional mapping hooks.
- Excluded all raw surveillance data and generated private analysis products.
